Includes: All 48 continental states + DC as individual SVG paths with CSS classes (e.g., .ca, .tx), plus all state border lines as separate stroke paths. Alaska and Hawaii are included but positioned separately.

How to Use

  1. Download the full SVG from the Wikimedia link above (~57KB)
  2. The SVG contains two main groups:
    • <g class="state"> - filled state shapes (classes like .al, .ca, .tx)
    • <g class="borders"> - state border lines (classes like .al-ga, .ca-nv)
  3. Restyle using CSS: change .state fill, .borders stroke, or target individual states
  4. Overlay the marker <g id="biodiversity-markers"> group on top
  5. Remove Alaska (.ak) and Hawaii (.hi) paths if not needed, or keep for context

Alternative: D3.js + TopoJSON Approach

For a programmatic approach, use us-atlas (npm) with d3-geo:

import {feature} from 'topojson-client';
import {geoAlbersUsa, geoPath} from 'd3-geo';
import us from 'us-atlas/states-10m.json';

const projection = geoAlbersUsa().scale(1070).translate([480, 300]);
const path = geoPath(projection);
const states = feature(us, us.objects.states);
